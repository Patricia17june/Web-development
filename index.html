Afro Fem Coders

About us

The afrofemcoders launched by Gloria Tumushabe aims to create a space that gives women a chance to learn programming.

Afro fem coders announced a partnership with the mastercard foundation, to inspire and empower African women with computer skills.

Afro Fem Coders is a program that was founded in March 2020 to help Ugandan women learn coding during the pandemic and thereafter.
Today, the program has students across Sub-Saharan Africa with the majority in Uganda.

Languages include

  • CSS
  • JavaScript
  • React
